api.chown.me documentation

/ip

— GET

Returns a JSON with the IP address.

/ua

— GET

Returns a JSON with the User-Agent field.

/post

RESTRICTED: It requires authentification through Basic Auth

— POST

Returns a JSON with the data from a HTML post form.

It also appends the header X-OTHER and X-OTTER.

/v1/irc

RESTRICTED: It requires authentification through Basic Auth

— GET

Returns a JSON with the data to be posted.

— POST

It needs in a HTML form:

  • chan: the chan where you want to post
  • message: the message you want to post
